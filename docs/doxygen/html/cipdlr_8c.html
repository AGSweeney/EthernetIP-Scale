Detailed Description

Implements the DLR object.

Author
Stefan Maetje stefa.nosp@m.n.ma.nosp@m.etje@.nosp@m.esd..nosp@m.eu

CIP DLR object

This module implements the DLR object for a non supervisor and non gateway device.

Implemented Attributes

  • Attribute 1: Network Topology
  • Attribute 2: Network Status
  • Attribute 10: Active Supervisor Address
  • Attribute 12: Capability Flags

Non-implemented attributes

The attributes 3, 4, 5, 6, 7, 8, 9 and 11 are only required for devices that are capable of functioning as a ring supervisor. These attributes shall not be implemented by non-supervisor devices.

The attributes 13, 14, 15 and 16 are only required for devices that are capable of functioning as a redundant gateway. These attributes shall not be implemented by non-redundant gateway devices.

None of the attributes 17, 18 and 19 is required and implemented. Because of this the Object Revision stays on level 3 (see DLR_CLASS_REVISION).

Implemented Services

  • GetAttributesAll
  • GetAttributeSingle
