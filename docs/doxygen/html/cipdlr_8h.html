◆ CipDlrCapabilityFlags

enum CipDlrCapabilityFlags

Provide bit masks for the Capability Flags attribute (#12).

The kDlrCapAnnounceBased and kDlrCapBeaconBased capability flags are mutually exclusive but one of it must be set.

Enumerator
kDlrCapAnnounceBased 

Device is an announce based ring node.

kDlrCapBeaconBased 

Device is a beacon based ring node.

kDlrCapSupervisor 

The device is capable of providing the supervisor function.

kDlrCapRedundantGateway 

The device is capable of providing the redundant gateway function.

kDlrCapFlushTableFrame 

The device is capable of supporting the Flush_Tables frame.
