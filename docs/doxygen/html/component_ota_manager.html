OTA Manager

Overview

The OTA Manager component provides over-the-air firmware update capability for the ESP32-P4 device. It supports both file upload via web interface and URL-based downloads.

Features

  • File upload via web interface
  • URL-based firmware downloads
  • Automatic rollback on failure
  • Progress tracking
  • Partition management

Usage

#include "ota_manager.h"
ota_config_t config = {
.url = "http://example.com/firmware.bin",
.timeout_ms = 30000
};
esp_err_t err = ota_manager_start_update(&config);
