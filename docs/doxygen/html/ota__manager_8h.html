Detailed Description

Over-The-Air (OTA) firmware update manager.

Provides OTA firmware update functionality, including URL-based updates, binary data updates, and streaming updates.

esp_ota_handle_t ota_manager_start_streaming_update ( size_t expected_size)

Start streaming OTA update.

Starts a streaming OTA update that writes directly to the partition without double buffering. Use ota_manager_write_streaming_chunk() to write data chunks, then ota_manager_finish_streaming_update() to complete.

Parameters
expected_sizeExpected firmware size in bytes (for validation)
Returns
OTA handle on success, 0 on error
